# vue-flask-sqlalchemy

## A guide for myself to quickly recap about vue.js, flask and flask-sqlalchemy
A general discussion is here:
https://medium.com/@herbzhao/vue-js-flask-flask-sqlalchemy-6a85a67a7790

Usage:
For frontend:
```
cd frontend
vue ui
```

For backend:

```
cd backend
pip install pipenv
pipenv shell
pipenv install
```
